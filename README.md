# PRVW Website

Marketing website for PRVW built with React, TypeScript, Vite, and Tailwind CSS.

## Getting started

```sh
# Install dependencies
npm install

# Start local dev server
npm run dev

# Build for production
npm run build
```

## Tech stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui components

## Deployment

Deploy the production build output generated by `npm run build` using your preferred static hosting provider.


## Uptime monitoring

A scheduled GitHub Actions workflow is included at `.github/workflows/site-monitor.yml`.

- Runs every 5 minutes.
- Checks `SITE_URL` for an HTTP 2xx/3xx response.
- Automatically opens (or updates) an outage issue when the site is down.
- Automatically comments and closes the outage issue when the site recovers.

### Configure notifications

1. In GitHub repository **Settings → Secrets and variables → Actions → Variables**, set:
   - `SITE_URL` (example: `https://prvw.ai`)
   - `MONITOR_ASSIGNEE` (your GitHub username, optional)
2. Ensure you are watching the repo or assigned to alerts so GitHub notifies you when an outage issue is opened.
