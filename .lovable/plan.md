## Interactive Elements
- **Scroll-triggered animations**: Sections fade/slide in as user scrolls
- **Animated number counters**: Statistics animate when in viewport
- **Interactive service explorer**: Tabs/accordions for service details
- **Hover effects**: Cards scale and reveal additional information
- **Cursor effects**: Subtle glow following cursor on dark sections
- **Client logo carousel**: Smooth infinite scroll

## Design Specifications
- **Color Palette**: Deep teal (#1a3a3a), Lime accent (#c5f92c), White, Light gray
- **Dark mode default**: Matches brand aesthetic
- **Flowing curve graphics**: CSS-generated or SVG background elements
- **Responsive design**: Desktop, tablet, and mobile optimized
- **Typography**: Clean hierarchy with bold headlines and readable body text
